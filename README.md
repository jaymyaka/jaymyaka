### TheDataEngineX Platform

My flagship open-source project — a self-hosted data/AI platform built across 6 repos and published to PyPI.

```
Raw Data → Lakehouse (Bronze/Silver/Gold) → ML Training + MLflow
                                                      ↓
                                         Vector Search (Qdrant)
                                                      ↓
                                         AI Agents + RAG → API → UI
```

| Repo | What it does |
| --- | --- |
| [**dataenginex**](https://github.com/TheDataEngineX/DEX) · [![PyPI](https://img.shields.io/pypi/v/dataenginex?label=PyPI&color=blue)](https://pypi.org/project/dataenginex/) | Core framework: medallion pipelines, ML registry, observability, plugin system |
| [**datadex**](https://github.com/TheDataEngineX/datadex) | YAML-driven pipeline engine: connectors, transforms, quality gates, lineage |
| [**agentdex**](https://github.com/TheDataEngineX/agentdex) | AI agent runtime: memory, tool registry, multi-model LLM routing, audit trails |
| [**careerdex**](https://github.com/TheDataEngineX/careerdex) | Career intelligence: job matching, salary prediction, skill gap analysis, Airflow DAGs |
| [**dex-studio**](https://github.com/TheDataEngineX/dex-studio) | Cross-platform desktop UI: pipeline monitoring, ML management, data lineage |
| [**infradex**](https://github.com/TheDataEngineX/infradex) | IaC: Terraform, Helm, Ansible, ArgoCD, Prometheus, Grafana, Jaeger |
